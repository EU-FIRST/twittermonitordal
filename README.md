# Twitter Monitor Project #
Twitter Monitor is provides infrastructure for acquisition, preprocessing, data access and html charting of twitter data.
## Components ##
It comprises of three subprojects which provide the described functionality. They are located in the three subfolders as follows:
### Pump ###
Pump is a daemon program running in background taking care the database is constantly being updated with new data. The two main tasks are data acquisition (from another DB), preprocessing and storing data into Twitter Monitor DB.
### DAL ###
DAL = Data Access Layer takes care of providing WCF services (Windows Communication Foundation e.g. Ajax, Soap, …) for the *UI* part from the DB that was generated by *Pump*.
### UI ###
UI = User Interface takes the data from *DAL* and displays it with some javascript library.
